Quick Start in 2 Minutes

Follow these steps to get TIGEN running and see your first 3D scene.

Step 1: Clone & Setup

Clone the repo and start a local dev server.

$ git clone https://github.com/terminaljoint/TIGEN.git
$ cd TIGEN
$ python3 -m http.server 8000

Then open http://localhost:8000 in your browser.

Step 2: Understanding the File Structure

TIGEN comes with everything you need. Here's what each file does:

main.js ‚Üê Entry point (initializes engine)
ecs.js ‚Üê Entity-Component System core
rendering.js ‚Üê 3D renderer with PBR support
physics.js ‚Üê Physics engine with collisions
audio.js ‚Üê Spatial 3D audio system
scene.js ‚Üê Scene graph & management

Step 3: Create Your First Scene

Create an HTML file and load TIGEN:

<!DOCTYPE html>
<html>
<head>
<title>My TIGEN Project</title>
<script src="https://cdn.jsdelivr.net/npm/three@0.145/build/three.min.js"></script>
</head>
<body>
<div id="app" />
<script type="module">
import { Engine } from './main.js';
const engine = new Engine();
engine.run();
</script>
</body>
</html>
